# TaskbarNumbers

Windowsのタスクバーに並んでいるアプリアイコンの上に「1〜9、0」の数字をオーバーレイ表示するPowerShellスクリプトです。
Windows標準のショートカットキーである「`Winキー + 数字キー`」によるアプリの起動や切り替えを、直感的に行えるようにサポートします。

## 特徴
- **視覚的なガイド**: 「あのアプリは何番だっけ？」と数える手間を省き、ショートカットキーを使いやすくします。
- **クリック貫通**: 数字が表示されていても、通常通りタスクバーのアイコンをマウスクリックできます。
- **リアルタイム追従**: アプリの起動・終了・並び替えに自動で追従します。スタートボタンや検索などのシステムアイコンは賢く除外されます。
- **バックグラウンド動作**: タスクトレイに常駐し、作業の邪魔になりません。

## 導入と起動方法

### 1. ショートカットの作成（黒い画面を出さずに起動する）
PowerShellスクリプト（`.ps1`）を直接実行すると一瞬黒い画面が表示されたり、コンソールが残り続けたりするため、バックグラウンドで静かに起動するためのショートカットを作成します。

1. クローンしたフォルダ内の `TaskbarNumbers` フォルダを開き、`TaskbarNumbers.ps1` を右クリックして「ショートカットの作成」を選択します。
2. 作成されたショートカットを右クリックし、「プロパティ」を開きます。
3. 「ショートカット」タブの「リンク先(T)」を以下のように書き換えます（元のパスの前後に実行コマンドを追加します）。

   ```text
   powershell.exe -ExecutionPolicy Bypass -WindowStyle Hidden -File "あなたの環境のパス\TaskbarNumbers.ps1"
   ```
   ※ `"あなたの環境のパス\TaskbarNumbers.ps1"` の部分は、クローンした実際のパスに置き換えてください（例: `"C:\Users\nhide\git\myPowerToys\TaskbarNumbers\TaskbarNumbers.ps1"`）。
4. 「適用」→「OK」を押して保存します。

これで、作成したショートカットをダブルクリックするだけで、画面に何も出さずにツールを起動できます。

### 3. スタートアップに登録してPC起動時に自動実行する
PCを起動するたびに自動的にこのツールを実行させたい場合は、以下の手順でスタートアップに登録します。

1. キーボードの `Win + R` キーを押して「ファイル名を指定して実行」を開きます。
2. `shell:startup` と入力して「OK」をクリックします。
3. スタートアップフォルダが開くので、先ほど作成した「黒い画面を出さずに起動するショートカット」をこのフォルダ内に移動（またはコピー）します。

これで、次回のWindows起動時から `TaskbarNumbers` が自動的に実行されるようになります。

## 終了方法
タスクトレイ（画面右下の時計やスピーカーアイコンがある場所）にあるインフォメーションアイコン（TaskbarNumbers）を右クリックし、「終了」を選択してください。
隠れている場合は、「^」アイコンをクリックして表示させてください。

# Mouse Jump

画面上の任意の位置にマウスポインターを素早く移動させるためのPowerShellスクリプトです。

## 機能

- 起動すると、全画面の縮小プレビューが表示されます。
- プレビューウィンドウ上の任意の場所をクリックすると、対応する画面上の位置にマウスポインターがジャンプします。
- マルチモニター環境にも対応しています。

## 使い方

1. `bin` フォルダにある `mouse_jump.bat` を実行します。
2. 画面にプレビューウィンドウが表示されます。
3. 移動したい場所をプレビューウィンドウ上でクリックします。
4. マウスポインターが移動し、ウィンドウが閉じます。

プレビューウィンドウは `Esc` キーを押すことでも閉じることができます。

## 素早く起動する方法 (ショートカットキー)

Windowsの標準機能だけで、キーボードから一瞬で起動できるように設定できます。

1. `bin` フォルダにある `mouse_jump.bat` を右クリックし、「ショートカットの作成」を選択します。
2. 作成された `mouse_jump.bat - ショートカット` を右クリックし、「プロパティ」を開きます。
3. 「ショートカット」タブの「ショートカット キー(K):」欄をクリックします。
4. 割り当てたいキーの組み合わせ（例: `Ctrl + Alt + M`）を実際にキーボードで押します。
5. 「OK」をクリックして保存します。

これで、設定したショートカットキーを押すだけで、いつでも `mouse_jump` を起動できます。

# Mouse Jump

画面上の任意の位置にマウスポインターを素早く移動させるためのPowerShellスクリプトです。

## 機能

- 起動すると、全画面の縮小プレビューが表示されます。
- プレビューウィンドウ上の任意の場所をクリックすると、対応する画面上の位置にマウスポインターがジャンプします。
- マルチモニター環境にも対応しています。

## 使い方

1. `bin` フォルダにある `mouse_jump_silent.vbs` をダブルクリックして実行します。
2. 画面にプレビューウィンドウが表示されます。
3. 移動したい場所をプレビューウィンドウ上でクリックします。
4. マウスポインターが移動し、ウィンドウが閉じます。

プレビューウィンドウは `Esc` キーを押すことでも閉じることができます。

## 素早く起動する方法 (黒い画面なし)

バッチファイル実行時に一瞬表示される黒い画面（コマンドプロンプト）を完全に表示させず、キーボードから一瞬で起動する方法です。

1. `bin` フォルダに `mouse_jump_silent.vbs` というVBScriptファイルが用意されています。
2. この `mouse_jump_silent.vbs` を右クリックし、「ショートカットの作成」を選択します。
3. 作成されたショートカットファイルを右クリックし、「プロパティ」を開きます。
3. 「ショートカット」タブの「ショートカット キー(K):」欄をクリックします。
4. 割り当てたいキーの組み合わせ（例: `Ctrl + Alt + M`）を実際にキーボードで押します。
5. 「OK」をクリックして保存します。

これで、設定したショートカットキーを押すだけで、黒い画面を一切表示せずに `mouse_jump` を起動できます。
